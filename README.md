# Student Management System (Flask)

A simple web application to manage **students, courses, attendance, and reports**.  
Built using **Python**, **Flask**, and **SQLite**. Great for practicing full-stack development with Python!

---

## Features

- **Dashboard:** Shows total students, courses, attendance, and active students.  
- **Students:** Add, view, edit, delete students and export data to Excel.  
- **Courses:** Manage course details easily.  
- **Attendance:** Record and track attendance.  
- **Reports:** Generate reports for specific dates with charts and export options.  

---

## Tech Used

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **Frontend:** HTML, CSS, JavaScript  
- **Charts:** Chart.js  
- **Export:** CSV/Excel  

---

## How to Run

1. Clone the repo:
bash
git clone https://github.com/ThanmaiReddy07/StudentManagementSystem.git
3. Install dependencies:
bash
pip install -r requirements.txt
4. Run the app:
bash
python app.py
